Sliding Window Pattern

Master efficient algorithms for array and string problems

Introduction to Sliding Window Pattern

The sliding window pattern is a computational technique used to solve problems involving arrays or strings. It involves maintaining a subset of elements (the "window") that slides through the data structure to find optimal solutions.

Key Concept: The sliding window technique is particularly useful for problems that ask for finding a subarray or substring that satisfies certain conditions.

When to Use Sliding Window

  • Finding the longest/shortest subarray with a given sum
  • Finding the longest substring with k distinct characters
  • Finding the maximum/minimum sum of a subarray of size k
  • Problems involving contiguous subarrays or substrings

Types of Sliding Windows

1. Fixed Size Window

Used when the window size is predetermined.

// Example: Find maximum sum of subarray of size k function maxSumSubarrayOfSizeK(arr, k) { let maxSum = 0; let windowSum = 0; // Calculate sum of first window for (let i = 0; i < k; i++) { windowSum += arr[i]; } maxSum = windowSum; // Slide the window for (let i = k; i < arr.length; i++) { windowSum = windowSum - arr[i - k] + arr[i]; maxSum = Math.max(maxSum, windowSum); } return maxSum; }

2. Variable Size Window

Used when the window size needs to be determined dynamically.

// Example: Find smallest subarray with sum >= target function smallestSubarrayWithGivenSum(arr, target) { let windowSum = 0; let minLength = Infinity; let windowStart = 0; for (let windowEnd = 0; windowEnd < arr.length; windowEnd++) { windowSum += arr[windowEnd]; while (windowSum >= target) { minLength = Math.min(minLength, windowEnd - windowStart + 1); windowSum -= arr[windowStart]; windowStart++; } } return minLength === Infinity ? 0 : minLength; }

Common Problems

1. Maximum Sum Subarray of Size K

Problem: Given an array of positive numbers and a positive number 'k', find the maximum sum of any contiguous subarray of size 'k'.

2. Smallest Subarray with Given Sum

Problem: Given an array of positive numbers and a positive number 's', find the length of the smallest contiguous subarray whose sum is greater than or equal to 's'.

3. Longest Substring with K Distinct Characters

Problem: Given a string, find the length of the longest substring in it with no more than k distinct characters.

Time and Space Complexity

  • Time Complexity: O(n) - Each element is visited at most twice
  • Space Complexity: O(1) - Only a constant amount of extra space is used
Important: The sliding window technique is most effective when the problem involves finding optimal subarrays or substrings with specific properties.
